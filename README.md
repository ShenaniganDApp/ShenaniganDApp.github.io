# Shenanigan Website

The Shenanigan website written with react

## How to run local

1. Clone onto your machine

    `git clone *repo_url*`
  
2. Install packages with yarn

    `yarn`
  
3. Run react start script

    `yarn start`

Congrats! It's all written in react so it's that simple!
